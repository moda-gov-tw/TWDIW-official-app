## [0.0.1-SNAPSHOT] - 2025-11-21

### Added
- 初始開源版本釋出 / Initial open source release
- 支援 W3C VCDM 1.1 (Verifiable Credentials Data Model v1.1) 協議 / Support for W3C VCDM 1.1
- 可驗證憑證 (VC) 驗證 / Verification of Verifiable Credentials
- 可驗證展示 (VP) 驗證 / Verification of Verifiable Presentations
- SD-JWT (部份揭露機制) 支援 / SD-JWT (selective disclosure) support
- 憑證狀態列表檢查 / Credential status list checking

### Security
- JWT 簽名驗證 / JWT signature verification
- 憑證有效性檢查 / Credential validity checking
- 憑證持有證明 (PoP) 支援 / Proof of Possession Support

### Documentation
- 新增 README.md 說明文件 / Added README.md documentation
